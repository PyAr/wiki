= Python Bug day (Junio 2008) =
== ¿De qué se trata? ==
Básicamente es una fecha prefijada donde programadores de todo el mundo se encargan de resolver bugs de Python. Mas información [http://wiki.python.org/moin/PythonBugDay aquí]
