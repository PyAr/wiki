Bloquear Click Del Mouse

CÃ³mo bloquear el click izquierdo del mouse cuando se tipea en el teclado, usando Python, en Linux, ejemplo simple, requiere Privilegios elevados en el equipo.

#!/usr/bin/env python
# -*- coding: utf-8 -*-
import os
import time
#
kb = file('/dev/console')
while True:
    os.system('xmodmap -e "pointer = default"') # Aca lo activamos
    kb.read(3)
    os.system('xmodmap -e "pointer = 0 2 3"') # Aca lo desactivamos
    time.sleep(0.5) # duerme, prueba reducir este valor para mas reaccion

Ejemplo:

sudo /usr/bin/env python blockclick.py
